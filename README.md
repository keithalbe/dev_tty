# dev_tty

My personal terminal and shell setup for a clean, reproducible dev environment.

## What it does

- Verifies `zsh` is installed
- Installs Oh My Zsh if missing
- Links:
  - `~/.zshrc`
  - `~/.gitconfig`

Everything else is sourced directly from this repo.

## Install

```bash
git clone https://github.com/keithalbe/dev_tty.git ~/.dev_tty
cd ~/.dev_tty
./install.sh
```
